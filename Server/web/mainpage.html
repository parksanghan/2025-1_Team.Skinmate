SKINMATE

당신의 피부를 과학적으로 관리하는 AI 기반 분석 서비스

앱 소개

SKINMATE는 인공지능 기반 피부 상태 분석 앱입니다.
셀카 또는 얼굴 이미지를 업로드하면, AI가 수분, 주름, 탄력, 모공 상태 등을 분석하고
맞춤형 피부 개선 솔루션을 제공합니다.

주요 기능

  • ✔ 얼굴 이미지 업로드 및 촬영 지원
  • ✔ 수분, 주름, 탄력, 모공 등 주요 항목 자동 분석
  • ✔ 분석 결과 기반 맞춤형 응답 및 추천 제공
  • ✔ 진단 이력 자동 저장 및 조회 가능

이용 방법

앱을 실행한 후, 얼굴을 촬영하거나 이미지를 선택하여 업로드하세요.
분석은 수초 내에 완료되며, 결과는 직관적인 수치와 설명으로 제공됩니다.

다운로드

Google Play 스토어에서 “SKINMATE”를 검색해 설치하세요.
또는 앱 배포가 완료되면 제공될 설치 링크를 통해 직접 다운로드하실 수 있습니다.
